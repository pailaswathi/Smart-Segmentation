Customer segmentation is an important task for businesses to understand different types of customers. In this project, we apply unsupervised machine learning techniques to group customers with similar characteristics.

The project analyzes customer data based on Age, Annual Income, and Spending Score and uses clustering algorithms to identify different customer segments.

Objectives:

Analyze customer data using exploratory data analysis (EDA).

Identify patterns in customer purchasing behavior.

Apply clustering algorithms to group similar customers.

Determine the optimal number of clusters using the Elbow Method.

Generate meaningful customer segments for marketing strategies.
